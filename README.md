# Documentation

## Run Locally

Clone the project

```bash
  git clone https://github.com/e4ebin/fast_api
```

Go to the project directory

```bash
  cd fast_api
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start the server

```bash
  uvicorn main:app --reload
```
